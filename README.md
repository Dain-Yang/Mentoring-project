# Mentoring-project
## 🚀 온라인 멘토링 매칭 플랫폼 (Online Mentoring Matching Platform)

> 다양한 주제와 경력에 기반하여 멘토와 멘티를 효율적으로 연결해주는 실시간 온라인 멘토링 매칭 플랫폼입니다.

### 🌟 주요 기능

본 플랫폼은 사용자 관리, 멘토 모집, 매칭 진행, 그리고 실시간 채팅 기능을 중심으로 구축되었습니다.

#### 1. 사용자 관리

* **기본 회원 기능:**
    * **회원가입, 로그인/로그아웃**, 기본 **정보 수정** 기능을 제공합니다.
* **역할 관리:**
    * 사용자를 **멘토**와 **멘티**로 명확히 구분하며, 역할에 따라 **API 접근 권한**을 제어합니다.
* **마이페이지:**
    * 사용자가 현재 **참여/진행 중인 멘토링 목록**, **신청 현황**, 그리고 **멘토링 완료 이력**을 조회할 수 있습니다.

#### 2. 멘토 모집 게시글 및 검색

* **게시글 관리 (멘토 전용):**
    * 멘토는 자신의 멘토링을 홍보하기 위한 모집 글을 **작성, 목록 조회, 상세 조회, 수정, 삭제(CRUD)** 할 수 있습니다.
* **카테고리/태그 관리:**
    * **기술 스택, 경력, 멘토링 주제** 등의 태그를 등록하고 조회할 수 있어 검색의 정확도를 높입니다.
* **멘토링 검색 및 필터링:**
    * **제목, 내용, 태그**를 포함하는 강력한 **검색 및 필터링** 기능을 제공하여 원하는 멘토링을 쉽게 찾을 수 있습니다.
* **모집 상태 관리:**
    * 게시글의 상태를 `모집 중` 또는 `매칭 완료`로 변경하고 마감 처리할 수 있습니다.

#### 3. 매칭 및 진행

멘토와 멘티 간의 멘토링 진행 과정을 체계적으로 관리합니다.

* **신청 상태 관리 (3단계):**
    1.  **신청 접수:** 멘티가 멘토링 신청서를 제출합니다.
    2.  **매칭 승인:** 멘토가 신청서를 검토 후 최종 승인합니다. **(승인 시 멘토링 채팅방이 자동 생성되고 정식 멘토링으로 전환됩니다.)**
    3.  **매칭 거절:** 멘토가 신청을 거절합니다.
* **멘토링 상태 관리 (3단계):**
    1.  **진행중:** 매칭 승인 후 멘토링이 활발하게 진행 중인 상태입니다.
    2.  **종료 요청:** 멘토 또는 멘티 중 한 명이 멘토링 종료를 요청한 상태입니다.
    3.  **멘토링 완료:** 양 당사자가 종료 요청에 최종 확인하여 멘토링이 공식적으로 마무리된 상태입니다.
* **멘토링 평가:**
   * **리뷰 작성:** 멘토링이 완료된 건에 대해서만 평점 및 상세 리뷰 작성을 허용합니다.
   * **평점 집계:** 멘토의 전체 리뷰를 기반으로 평균 평점을 계산하고, 이를 멘토 프로필에 노출하여 신뢰도를 제공합니다.

#### 4. 실시간 채팅

* **채팅방 목록 조회:** 참여 중인 멘토링의 **채팅방 목록**을 조회할 수 있습니다.
* **실시간 채팅 및 채팅방 생성:**
    * **WebSocket**을 활용하여 멘토와 멘티 간의 **실시간 메시지 송수신**이 가능합니다.
    * 멘토가 `매칭 승인` 시 멘토링 채팅방이 **자동 생성**되며, 멘토링 완료 후에는 **잠금 처리**됩니다.

### ✨ 추가 기능 (3순위)

플랫폼의 성능과 사용자 경험을 향상시키기 위한 추가 기능입니다.

#### 1) 평가 시스템 성능 개선

* **별점 캐싱:** 평점 입력 시 **Redis**를 활용하여 즉시 캐시 처리합니다.
* **상세 리뷰 비동기 처리:** 상세 리뷰는 시스템 부하를 줄이기 위해 **비동기 방식**으로 처리하여 서비스 안정성을 확보합니다.

#### 2) 검색 최적화

* **검색 엔진 도입:** 멘토링 검색 기능을 **MySQL의 `LIKE` 쿼리 대신** **Elasticsearch**를 이용하여 구현하여, **더 빠르고 정확한 전문 검색(Full-Text Search)** 성능을 제공합니다.
